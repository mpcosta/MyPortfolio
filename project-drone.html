ADDDS

Autonomous Drone Document Delivery System or ADDDS has started as part of a small Artificial Intelligence Planning Project using the PDDL Planning Language. The project aims to create an autonomous drone delivery system for primary use inside companies. This would be assisted by different Machine-Learning techniques enabling the drone to take the most efficient route during the process of delivery.

The small-scale prototype AI Planning related to this project shows that the planning works even if we had more than one drone involved and multiple documents to deliver to different locations.
