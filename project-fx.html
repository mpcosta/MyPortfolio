FX Throw Rush

FX Throw Rush is an Android Game created as part of my final high school year project. All the technologies, languages and methods involved in the creation of this project were self-taught.

The game was created from scratch using Unity Engine 4.0 along with JavaScript and C#. The game also implements different services for improving it's gameplay/quality and revenue such as Google Play Game Services, Google Play In App Purchase, Admob and Facebook API.
