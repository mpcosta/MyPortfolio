Adam

Adam is an application created for A-Level macro-economic students who need to access quick information about some country or region. This application displays information related to the searched economic indicators using the World Bank Data API.

Several extra features implemented such as quizzes that allow training on the different indicators and macro-economic data, speech recognition for simple queries and search suggestions based on what the user types.
