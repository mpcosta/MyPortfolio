Fresh

Fresh Movies is a small project about a Movie Trailer Website generated by a python server side client file. This website contains some features such as the ability of only supplying the movie name on the server side file and the software will automatically search for the closest match acquiring the movie poster and trailer.

This project was made entirely with the purpose of exploring the python language and learn some of its potential.

Fresh Main Interface

Enable user to interact with different movies.

Fresh Movie Trailer

When pressing on a movie it plays its respective trailer.

Fresh Auto Search

It can search for movie resources just by providing a class with the respective name.
